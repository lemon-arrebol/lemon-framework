# lemons agent
通用组件

MANIFEST.MF用于定义扩展和包相关数据的清单文件。 
注意事项： 
1. MANIFEST.MF中的数据是以key: value [value1 value2]形式存储 
2. 冒号必须是英语的冒号:,冒号后面要带有一个空格 
3. 如果value有多项要用空格隔开 
4. 文件最后必须要有一个空行
